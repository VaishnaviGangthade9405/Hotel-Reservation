# Project Overview
The Hotel Reservation Management System is a web-based application developed to simplify and automate hotel booking and management processes.
This system allows users to check room availability, make reservations, and manage bookings efficiently, while administrators can manage rooms, customers, and reservations through a secure dashboard.
This project demonstrates practical implementation of PHP, MySQL, and Web Technologies in building a real-world database-driven application.

#Objectives
-To automate the hotel room reservation process
-To reduce manual work and errors in booking management
-To provide an easy-to-use interface for both customers and administrators
-To apply database concepts like CRUD operations and relational design

#Target Users
Hotel Customers (for booking rooms)
Hotel Administrators (for managing reservations, rooms, and users)

#Key Features
-User registration and login
-View available rooms
-Book hotel rooms online
-View booking details

# Admin Features
-Secure admin login
-Add, update, and delete room details
-View all customer bookings
-Manage hotel reservation records

#Technology Stack
-Frontend: HTML, CSS, JavaScript
-Backend: PHP
-Database: MySQL
-Server: Apache (XAMPP)
-Tools: VS Code, Git, GitHub
